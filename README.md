## General info
This is like an e-commerce site. You can spend bill gates' money by buying various products.
	
## Technologies
Project is created with:
* JavaScript
* CSS
* react: 18.2.0
* redux: 4.2.0
* react-redux: 8.0.2
* reduxjs/toolkit: 1.8.5
* react-countup: 6.3.0

	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
